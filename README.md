# 全渠道AI人文与纪实创作者追踪中心

跨平台（抖音 / B站 / 小红书）AI人文、AI纪实、人文纪实类创作者监测报表。

## 访问地址

部署后通过 GitHub Pages 访问：
```
https://你的用户名.github.io/creator-tracker/
```

## 数据范围

- **平台**：抖音、B站、小红书
- **品类**：AI人文类、AI纪实类、人文纪实类
- **体量**：头部（500万+）、中腰部（1-500万）
- **预置创作者**：58位

## 本地使用

直接用浏览器打开 `index.html` 即可使用，支持：
- 按平台 / 品类 / 体量 / 状态筛选
- 搜索创作者、作品、备注
- 添加 / 编辑 / 删除创作者
- 导出 / 导入 JSON 数据

## 数据更新

1. 在页面内编辑创作者信息
2. 点击右上角「导出JSON」保存数据
3. 更新 `index.html` 后重新部署

## 定时任务

已配置每周日上午10点自动抓取各平台最新创作者动态，生成周报摘要。
